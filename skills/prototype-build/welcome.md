# Prototype Builder

Reads a PRD from the project directory, analyzes it, and generates a multi-file HTML prototype — one HTML file per screen — with embedded mock data, a floating State panel for role and scenario switching, functional interactive widgets in each screen, and role-based navigation via localStorage and href links.

You have **1 input** and **2 defaults** you can configure. *(Say "show inputs" or "change defaults" anytime.)*
